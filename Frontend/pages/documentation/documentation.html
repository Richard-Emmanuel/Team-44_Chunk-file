Introduction

Chunkeet is a platform that accepts large CSV and JSON files and breaks them into smaller bits while maintaining the right format. The platform allows authenticated users, to upload their large files, chunk it into bits and download to their local device. Team 44 has been able to solve most issues related to splitting of CSV and JSON files. Issues like; chunked files inaccuracy, data incompletion and system errors etc.

what is a JSON File

JSON is an open standard file format for sharing Data that uses human-readable text to store and transmit data. It is a general data format used with different applications, including web applications. It is a lightweight format for storing and transporting data. Web Developers use JSON to transmit data from the server to the web browser and from =the web browser back to the server.
